Each track is a sequenced set of lessons aimed at a specific audience. Pick the one that matches where you are.

## Track 1: Getting Started with Clawless

For readers brand-new to Clawless. Install, first chat, first agent, all in 30 minutes. The fastest path to your first "I built something" moment.

*Status: lessons coming soon (Phase 1, in progress).*

## Track 2: Use Case Cookbook

"I want to do X" recipes for research, email triage, code review, meeting notes, travel planning, and more. Copy-pasteable, runnable in Clawless.

*Status: Phase 3.*

## Track 3: Agent Building 101

Skills, tools, channels, cron. The extension layer of Clawless, explained without jargon.

*Status: Phase 6+.*

## Track 4: OpenClaw Deep Dive

For developers and power users. Engine internals, MCP, custom providers. Goes where the product docs don't.

*Status: Phase 6+.*

Transformers, attention, agents, RAG, tool use. Platform-neutral at the conceptual level, for readers who want to understand how it works, not just use it.

Why local. Vault. Key management. Threat models. For the privacy-conscious reader.
